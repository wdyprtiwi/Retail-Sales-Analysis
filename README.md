# Retail-Sales-Analysis
Designing and developing an interactive retail sales dashboard to provide insights into sales performance and customer trends.
## Column
<ol>
  <li>Age</li>
  <li>Customer ID</li>
  <li>Date</li>
  <li>Gender</li>
  <li>Price per Unit</li>
  <li>Product Category</li>
  <li>Quantity</li>
  <li>Total Amount</li>
  <li>Transaction ID</li>
</ol>

## Insight
<p>The goal of this project is to analyze data from a retail sales records and answer the questions beneath:</p>
<ol>
  <li>How much is the total sales in 1 year?</li>
  <li>Which product generates the highest sales?</li>
  <li>How many items are sold in each product category?</li>
  <li>How many items are sold in total each month?</li>
</ol>
